# Support

## Where to get help

| Question | Place |
|----------|--------|
| Croc GUI UI, packaging, crashes, installers | [Open an issue here](https://github.com/interfluve-wav/croc-gui/issues) |
| How croc transfers work, relay/CLI bugs | [schollz/croc issues](https://github.com/schollz/croc/issues) |
| Feature ideas for this desktop app | [Feature request](https://github.com/interfluve-wav/croc-gui/issues/new?template=feature_request.yml) |

Croc GUI is an independent wrapper. Supporting upstream keeps the engine healthy for everyone.

## Before filing a GUI bug

1. Confirm you are on a recent [release](https://github.com/interfluve-wav/croc-gui/releases) or `main` build.
2. Note OS, arch, GUI version (About), and bundled croc version (About).
3. Say whether **local-only**, custom relay, or zip options were enabled.
