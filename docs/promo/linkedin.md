Secure file transfer shouldn't require a terminal — or uploading to someone else's cloud.

I built Croc GUI: a free, open-source desktop app for schollz/croc — the popular encrypted peer-to-peer file transfer tool.

What you get:
→ Drag-and-drop send and receive on macOS, Windows, and Linux
→ QR codes and code phrases for easy hand-off between devices
→ Local-only LAN mode for air-gapped or home-network transfers
→ No accounts, no vendor cloud — files go directly peer-to-peer

Try it or star the repo: https://github.com/interfluve-wav/croc-gui

## Posting tips

| Tip | Detail |
|-----|--------|
| **Timing** | Tue–Wed, 8–10am in your network's timezone |
| **Image** | Posts with images get more impressions — use `banner-community.png` |
| **Link** | Put link once at end; avoid link-only posts |
