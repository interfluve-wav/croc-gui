## Optional thread (reply to main tweet)

**Tweet 2 — what it is**

```
What is it?

A Tauri 2 + React desktop front end that shells out to a bundled croc binary. Same end-to-end encrypted transfers — just a clear Send / Receive UI instead of memorizing CLI flags.
```

**Tweet 3 — features**

```
Highlights:
• Code phrase + QR for hand-off to phone/second machine
• Custom relay, port, overwrite, auto-confirm
• Local-only (--local) for LAN peers
• Preferences for download folder + default relay
• Prebuilt installers on GitHub Releases
```
