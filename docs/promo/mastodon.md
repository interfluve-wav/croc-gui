## Posting tips

- Post when your home instance is active (evenings US/EU often fine).
- Use `Content warning` only if your instance expects CW on self-promo — check local norms.
- Boost from any project/org account you run; ask friendly mutuals to boost (not beg).
- Link GitHub Releases directly when announcing a version.
